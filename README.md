# ?? ProcureAI UAE - Comprehensive Analytics Dashboard

A complete machine learning dashboard for customer segmentation and dynamic pricing in the procurement SaaS industry.

## ?? Features

### 1. Classification Analysis
- Predict customer purchase intent (Hot Leads)
- Multiple classification models
- ROC curves and confusion matrices
- Feature importance analysis

### 2. Clustering & Segmentation
- K-Means clustering with interactive k selection
- Elbow method and silhouette analysis
- PCA visualization
- Customizable persona naming
- Cluster profile comparison

### 3. Association Rule Mining
- Apriori algorithm implementation
- Adjustable support, confidence, and lift thresholds
- Interactive rule exploration
- Top N rules display
- Visual analysis of rule metrics

### 4. Regression Analysis
- Predict willingness to pay (WTP)
- Multiple regression models
- R² and RMSE metrics
- Feature importance
- Actual vs Predicted visualizations

### 5. Dynamic Pricing Engine
- Automated tier assignment
- Personalized discounts
- Revenue projections
- Customer prioritization
- Annual vs monthly pricing
